Heuristic Evalution and Changes Report

We collected a wide variety of heuristic evaluation notes from our peers ranging from small UI tweaks to major bug fixes. This report contains the modifications made to the Impact app after evaluating the received issues.


Changes

1: Background color was given to the “View Donations” button on the main page of the Impact App. This was done to give the button more affordance for the user, to let them know that this is a button to be clicked. It is also consistent with the color of other buttons within the app.

2: In the “recent transactions” tab, names of each of the views were changed. “List View” was changed to “Recent.” “Graph View” was changed to “History.”

3: Replaced the Humane Society image with the ASPCA image on the top of the home screen indicating that the user is currently donating to the ASPCA. This was done to indicate more clearly that this space shows which organization the user is currently donating to.

4: Fixed the back button to actually go back to the previous page, rather than go to the home screen.

5: Added “View Donations” to the navigation bar list menu. This gives the user the ability to access all functionality of the app from the navigation bar list menu, instead of just categories from before.

6: Changed the “Donate” text to “Start Donating”. Users were confused by the word “Donate,” as they thought that clicking this button meant they were going to make an immediate donation.

7: Changed the “Thank you for donating!” text after the user presses “Start Donating” to “Thank you!” Similar to the above point, this removes any confusion of immediately making a donation at that very moment.
